# Windhawk PERSONAL Profiles (for me, sb4ssman)

Saved Windhawk modset + settings snapshots, one subfolder per machine or configuration.

Each subfolder contains whatever the backup tool exports — typically a list of enabled mods
and their settings. Name folders by machine or role (e.g., `desktop-main`, `laptop-work`).

Backup tool: [Windhawk-Services-Backup-Utility](https://github.com/scorpion421/Windhawk-Services-Backup-Utility)
